MDes Peer Program Atlas

The MDes Peer Program Atlas is a browser for exploring and comparing peer programs in the corpus—how they describe identity, positioning, duration, cost, and curriculum, drawn from public materials. The data is assembled for research and discussion, so it is necessarily imperfect; if you spot gaps or errors, edits are welcome. Open any row for details, click Edit, change what you need, then use Export patch for this program to download a JSON file you can send to the admins for review.
